🥗 NutriLens – AI Food Recognition & Nutrition Tracker
🔍 Problem

One of the biggest challenges in maintaining a healthy lifestyle is accurately tracking food intake. Most existing calorie-tracking apps rely on manual input, which is time-consuming, error-prone, and discourages consistent usage.

I wanted to build a system that could:

Automatically identify food items from images
Estimate nutritional values
Reduce manual effort in tracking diet
💡 Solution

I built NutriLens — a mobile application that uses computer vision to detect Indian food items from images and estimate their nutritional content.

⚙️ What I Did
Developed a custom YOLOv8 model trained on Indian food datasets
Created a dataset of 1100+ images using Roboflow and applied augmentation techniques
Used transfer learning to improve model performance
Achieved 89.4% mAP@50 accuracy, ensuring reliable predictions
🏗️ System Architecture
User captures image via mobile app
Image sent to backend (Flask server)
YOLO model processes image and detects food items
Detected items mapped to calorie database
Results returned and stored
🧩 Backend & Features
Firebase integration for user data
Google OAuth authentication
Streak tracking system
Analytics dashboard for user insights
🧠 Key Learnings
Building ML systems is not just about models — data quality matters more
Real-world accuracy requires iteration and tuning, not just training once
Integrating ML with applications requires careful pipeline design
